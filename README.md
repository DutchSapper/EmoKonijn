# EmoKonijn

<p>Emokonijn is a application i did as a school project, it translates (dutch > english) to Emoticon Rabit. The main focus was on the use of arrays, functions and objects.</p>
<p>When de application starts it will present two lines of 15 random emoticon rabits. When the rabit on the first and second line match the score below will be updated and will be saved in there own file in the folder "score".
Everytime you refresch the page the rabits will jump to a new place on the lines and the score will jump with the first line rabit. </p>

<p>The most dificult thing was creating the arrays so i could use it for random rabits but also for getting the score and the text files right.</p>
